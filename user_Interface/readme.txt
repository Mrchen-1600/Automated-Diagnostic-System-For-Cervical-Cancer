本目录下文件说明：
    config: 目录下存放的是UI.py中的相关配置参数，在启动UI.py之前，需要先修改config目录下的config.py文件
    iamges: 目录下存放的是图片文件，包括UI展示图片、待识别的细胞图像集、临时文件夹
        test_imgs: 目录下存放的是待识别的细胞图像集
        tmp: 目录下存放的是临时图片，当单个样本识别选择了识别图像后，封面图会替换成细胞样本图像，对应图像就存储在该文件夹下
        UI：目录下存放用于UI展示的图片
    reports: 目录下存放的是识别报告，无论是否下载，每次识别都会自动保存一份在此目录下
    resnet50: 目录下存放的是resnet50的模型文件、预测文件和训练好的权重文件
    vgg16: 目录下存放的是vgg16的模型文件、预测文件和训练好的权重文件
    visiontransformer: 目录下存放的是visiontransformer的模型文件、预测文件和训练好的权重文件
    UI.py: 展示UI界面
